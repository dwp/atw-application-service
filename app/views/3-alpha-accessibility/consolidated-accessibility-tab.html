Accessible information about Access to Work

There is a range of formats with information about applying for Access to Work.

We use simple language and clear layouts for our online guidance. You can navigate this using speech recognition software and listen using a screen reader.

Information about applying for Access to Work is also available in these formats:

  • British Sign Language (BSL) videos with audio and subtitles
  • easy read
  • large print
  • braille
  • audio CD

If you need help

You can get help with your application by using the Access to Work helpline or email address. You can contact us in these ways:

  • telephone: 0800 121 7579
  • textphone: 0800 121 7579
  • Relay UK service, if you cannot hear or speak on the phone: 18001 then 0800 121 7479
  • video relay service for BSL users (find out how to use this service)
  • email: dwponline.helpdesk@dwp.gov.uk

These services are available from Monday to Friday, 9am to 5pm.

If you don't need support, you can continue to apply online.
